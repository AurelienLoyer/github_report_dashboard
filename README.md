## CONFIG

Duplicate config/config.model.js file with your config
- config/config.dev.js
- config/config.prod.js

```config.dev.js
  GITHUB_CLIENT_SECRET: '3789dsqdh567890dsqldj1234',
  BASE_URI: 'http://localhost:1337',
  PORT: 1337
}

```
